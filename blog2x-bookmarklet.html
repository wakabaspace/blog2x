🚀 Blog2X - ブックマークレット設定ガイド

1. ブックマークレットとは？

ブログ記事を開いた状態でクリックするだけで、記事の情報を自動取得してBlog2Xツールに渡すことができます。

💡 こんな時に便利：
  • 自分のブログ記事をXでシェアしたい
  • 他人のブログ記事を引用・紹介したい
  • キャッチーなポスト文を素早く作りたい

2. 設定方法

ステップ1: ブックマークバーを表示
  • Chrome/Edge: メニュー → ブックマーク → ブックマークバーを表示
  • Safari: 表示 → ブックマークバーを表示
  • Firefox: 表示 → ツールバー → ブックマークツールバー
ステップ2: ブックマークレットをドラッグ＆ドロップ

下のボタンをブックマークバーにドラッグしてください（クリックしないで！）：

📝 Blog2X

✅ 完了！ ブックマークバーに追加されたら、すぐに使えます。

3. 使い方

ステップ1: ブログ記事を開く

自分のブログでも、他人のブログでも、Xでシェアしたい記事を開きます。

ステップ2: ブックマークレットをクリック

ブックマークバーの「📝 Blog2X」をクリック。

ステップ3: 自動で情報が取得される

新しいタブでBlog2Xツールが開き、以下が自動入力されます：
✓ 記事タイトル
✓ 記事URL
✓ 記事の見出し・太字・本文
✓ 記事内の画像

ステップ4: ポスト候補を生成

「ポスト候補を生成」ボタンをクリックすると、複数パターンのポスト文が自動生成されます。

ステップ5: コピーしてXに投稿

気に入ったポスト候補の「コピー」ボタンをクリックして、Xに貼り付けて投稿！

4. トラブルシューティング

❌ ブックマークレットが動作しない

  • ポップアップブロック：ブラウザの設定で、新しいタブを開くことを許可してください
  • ドラッグ＆ドロップ失敗：ボタンをクリックせず、長押ししてドラッグしてください

❌ 情報が取得できない

  • 一部のブログ構造：特殊な構造のブログでは、自動取得がうまくいかないことがあります
  • 手動入力：その場合は、ツールを直接開いて手動で情報を入力してください

💡 より便利な使い方

  • 自分のブログ：記事を公開したら、すぐにブックマークレットでポスト作成
  • 他人のブログ：引用・紹介する際に、URL付きでキャッチーにシェア
  • 画像選択：複数の画像がある場合、一番目立つものを選んでポスト
🎉 設定完了！

これでブログ記事のXシェアがぐっと楽になります。
